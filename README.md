# youtube-transcribe

Universal audio/video transcription tool with 8 interchangeable backends — local Whisper (offline by default), YouTube subtitles, Gemini, Groq, OpenAI, Deepgram, AssemblyAI, and any OpenAI-compatible custom API.

> **🚧 Project in active development — implementation has not yet started.**
> The design and implementation plan are complete; code generation is the next step.
